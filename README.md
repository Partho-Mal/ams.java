# AMS.java Attendance Management System

## Description

A modernized Attendance Management System built in Java. It features refined logic, a cleaner UI flow, and extended functionality. The work takes inspiration from earlier open source projects while adding original changes for clarity, maintainability, and a smoother user experience. This project was created during my second year Mini Project for the B.E in Information Technology program.

## Tech Stack

* Language: Java (Swing, AWT)
* Database: MySQL and JDBC
* Libraries Used:

  * java.awt and javax.swing for the interface
  * java.sql for database integration
  * java.text.SimpleDateFormat for date handling

## Core Features

* Add, update, and delete attendance records
* Dynamic data display through JTable
* Integrated date and time tracking
* Simple and responsive interface
* Improved data handling and SQL interaction
* Additional quality of life features added during development

## Acknowledgment

This project is based on the idea of open source attendance systems that are available online. The structure and code were rewritten with original modifications to improve readability, flow, and reliability.
